Heart Disease Prediction System

Project Aim

The main purpose of this project is to predict whether a patient is of Heart Disease on not?


Input and output

Input: Pateint

Output: Prediction (Have Disease / Not Have Disease)


Representation of Input (Patient)

  Input (Patient) is Represented as 4 Input Attributes

  Input Attributes 01– Age (Patient Class)

    Possible Value 01 – Old

    Possible Value 02 – Mature


  Input Attributes 02– Sex

    Possible Value 01 – Male

    Possible Value 02 – Female


  Input Attributes 03– Cp

    Possible Value 01 – Zero

    Possible Value 01 – One

    Possible Value 02 – Two

    Possible Value 03 – Three


  Input Attributes 04– Chol

    Possible Value 01 – Zero

    Possible Value 02 – One


Representation of Output

   Output is Represented as a single Output Attributes

    Output Attribute – Target

    Possible Value 01 – Yes

    Possible Value 02 – No


Example - Input and Output

Input (Passenger):

  Age:

    Old

  Sex:

    Male

  Cp:

    Two

  Chol:

    Zero

Output: Not Have Disease
